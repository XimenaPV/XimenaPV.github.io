The Minimal HTML in VIM

The first thing we had to memorize was the minimal HTML page. This the base to your webpage. It always has the same format. This is the way to write it:

  1. Open the terminal on your computer.
  2. Access the directory where you want to have the html.
  3. Once you are in your directory, type "vim index.html". This will take you to the text editor where you will create your webpage.
  4. Now that you are in the text editor, type the html codeby, first, press I for insert and be able to add text.
  5. Then, type the following:
	
	<DOCTYPE html>
	<html lang="en">
	<head>
		<title><Homepage></title>
	</head>
	<body>
		<h1><Hello, World!></h1>
	</body>
		</html>

Here is a brief explanation of how this code works:

  • < and > : Tags work in pairs. HTML uses tags, so when you want to write basically anything make sure it has an opening "<" and a closing ">" so that it works.
  • <!DOCTYPE html> : This tag tells the browser the version of HTML. Always needed when writing HTML.
  • < html lang="en"> : html is the root element. It tells the browser where the HTML document begins and ends. lang indicates the language of the page and "en" is the language, which in this case it is english.
  • <head> : The opening tag for the element that contains the meta information about the HTML page. Everything inside this tag will not be directly showed in the page.
  • <title><Homepage></title> : The title element specifies a title for the html that shows in the page's tab.>Homepage is the first page you see when you open the browser.
  • </head> : This tag is just the closing one for the "<head>" tag.
  • <body> : Now, every tag you add inside the body will appear in the page.
  • <Hello, World> : This is the text that will be shown in the web page when you first open it.
  • </body> and </html> : These two tags are the closing tags for the body and html tags respectively.
